# Box

`Box` component can be used to pad it's children with margin, padding or size. All attributes that contain plain numbers (without units) get automatically spanned. Breakpoint codes get converted into pixels values in the attributes where it makes sense.

~~~
import { Box } from '@deliveryhero/armor';
~~~

## Basic usage

<Preview>
